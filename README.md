# Legal AI Website

A simple open-source legal AI web project that helps users understand legal topics, explore legal questions, and interact with an AI-powered legal assistant interface.

## Features
- Clean legal-tech landing page
- AI legal assistant interface
- Modern responsive design
- Open source and easy to extend

## Project Goal
This project aims to make legal information easier to access through a simple AI-powered web experience. It is designed as a starting point for building legal-tech tools for students, freelancers, and everyday users.

## Tech Stack
- HTML
- CSS
- JavaScript

## Roadmap
- Add real AI chat integration
- Add legal document summarizer
- Add contract explanation tool
- Add multilingual support
- Add FAQ and disclaimer system

## Installation
1. Clone the repo
2. Open `index.html` in your browser
